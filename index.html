QR Details Viewer

This page reads details from the scanned QR's URL and shows them below. Use copy buttons to copy each field.

Warning: embedding passwords in URLs/QR codes is insecure. Use short-lived tokens or server lookup for production.
Raw URL:

If nothing appears, the QR likely doesn't include `?fn=...&sn=...&email=...&pw=...` parameters. Example URL format shown below.

Example:
https://your-username.github.io/your-repo/?fn=Saif&sn=Ali&email=saif%40icloud.com&pw=MySecret123
